## Values

This list prioritizes:
- **FOSS** (FSF/EFF aligned)
- **Privacy & Data Sovereignty** (local-first, anti-surveillance)
- **Human-Made** (no AI-integrated tools)
- **Accessibility** (ergonomic, low cognitive load)
- **Ethical Companies** (non-14 Eyes jurisdictions when possible)

---

## Core System

| Category | Software | Notes |
| :--- | :--- | :--- |
