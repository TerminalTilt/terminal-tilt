# Terminal Tilt

Channel branding, assets, and video support documents for [Terminal Tilt](https://terminaltilt.com).

## Brand Identity

Terminal Tilt creates content about Linux, FOSS, privacy, and digital sovereignty.

**Tagline:** "Stay sovereign. Stay secure. Stay private."

## Assets

| Folder | Contents |
|--------|----------|
| `assets/logos/` | Channel logos (icon + full text) |
| `assets/banners/` | Social media banners (YouTube, GitHub, Mastodon) |
| `assets/colors/` | Color palette reference |

## Usage

Feel free to use these assets for:
- Fan content
- Video essays about Linux/FOSS
- Community projects

Please don't use for:
- Commercial products without permission
- Misrepresentation or impersonation
